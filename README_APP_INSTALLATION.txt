MISSION ZELTLAGER - INSTALLIERBARE WEB-APP

Diese Version ist eine PWA. Sie kann auf dem iPad über Safari zum Home-Bildschirm hinzugefügt werden.

Wichtig: iOS installiert PWAs nur, wenn die Dateien über einen Link/Webserver geöffnet werden, nicht direkt aus der Dateien-App.

Kurzweg:
1. Ordner/ZIP auf Netlify, GitHub Pages oder einen Schulserver hochladen.
2. Link auf dem iPad in Safari öffnen.
3. Teilen-Symbol antippen.
4. Zum Home-Bildschirm antippen.
5. Namen bestätigen: Mission Zeltlager.

Nach dem ersten Öffnen werden die Dateien offline gespeichert. Vor dem Geburtstag einmal auf jedem iPad komplett testen.
